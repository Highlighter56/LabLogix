# LabLogix
LabLogix is a real-time tracking app that allows students maximize laboratory use-time through the ability to view open seats in real-time.

## Run Quick Reference

### Interactive Terminal 

Windows:
1. Open terminal in project root.
2. Run: scripts\\run-terminal.cmd

macOS:
1. Open terminal in project root.
2. Run once: chmod +x scripts/run-terminal.sh
3. Run: ./scripts/run-terminal.sh

### Website Server

Windows:
1. Open terminal in project root.
2. Run: scripts\\run-website.cmd

macOS:
1. Open terminal in project root.
2. Run once: chmod +x scripts/run-website.sh
3. Run: ./scripts/run-website.sh
